# What is Unified Modeling UML?

1. UML stands for Unified Modeling Language.
2. UML is a graphical language for software visualizing, software specifying, software constructing, and documenting information about software system  as well as for business modeling and other non-software systems.
3. UML can be used for modeling a system independent of the plateorm language.


1. UML gives a standard way to write a system model, covering conceptuals ideas.

2. With an understanding of the modeling, the use and application of UML can make the software development process more efficient.

3. Some UML tools generate program language code from UML.

4. By using visual representations, we are able to better understand possible flows or errors in software or business processes

# What is Pseudocode ?
## ```Pseucodo is a plain language descriptions of the steps of algorithm. Pseudocode is for developer to write algorithms steps and understanding the logic```

1. Human language
2. Peusudocode Fake code not source code
3. No broad standard to writing  Pseudocode
4. Pseudocode is not and executable program.
---

## `Example`
```

step 1. Take first number in a , and second number in b

step 2. add a and b 

step 3. store a and b value in c

step 4. display result of c.

```
